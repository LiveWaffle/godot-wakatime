# Wakatime for GoDot
*Count and Track time in Godot*

### Prerequisites

* GoDot
* Git


### Installing

Clone the repository

```
git clone https://github.com/LiveWaffle/godot-wakatime.git
```

Open Godot and Drag it into res://Addons or use the asset library!
